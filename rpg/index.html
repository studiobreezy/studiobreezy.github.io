Hidden Peak: A Tale From Highspire

In development

Completed Systems & Features

  • Dialogue System — typing effect, speaker tags, player movement freeze
  • Inventory — stackable items, tooltips, use/drop functions, protected items
  • Day/Night Cycle — dynamic lighting synced with the game clock
  • Scene Transitions — with spawn points, scene fading, and additive loading
  • Mini-Game Integration — multiple playable arcade game
  • NPC Daily Schedules — pathfinding, persistent positioning, branching dialogue
  • Currency System — coins
  • Shop System — interactable NPC shopkeeper, purchase items from shop counters and shelves, rotating seed display
  • Save System — player progress, time of day, inventory, and more
  • Quest System — track and manage player quests

Currently Working On

  • Turn-Based Battles — strategic combat system with player and enemy turns, status effects, use items and special abilities
  • Character Creation — customize your character's appearance and attributes
  • Weather System — dynamic seasonal weather effects
  • Placeable Furniture — place, rotate, and pick up furniture items
  • Camping system — placeable tents and campfires from inventory
  • In-Game Desktop — a lore delivery system with access various minimal in-game applications
  • Mining — gather resources from a vast cave in Highspire
  • Crafting System — create items and tools from gathered resources from the player's workshop
  • Intro Story — intro cutscene and player waking in the town Inn
  • Worldbuilding — interior scenes (arcade, homes, shops, caves, and more), exterior locations to explore, town lore, Highspire’s history

Prototyping & Future Goals

  • Emotional story arcs tied to exploration, memory, and community
