Hidden Peak: A Tale From Highspire

In development

Completed Systems & Features:

  • Dialogue System — typing effect, speaker tags, player movement freeze
  • Inventory — stackable items, tooltips, use/drop functions, protected items
  • Day/Night Cycle — dynamic lighting changes, clock UI
  • Scene Transitions — with spawn points, scene fading, and additive loading
  • Mini-Game Integration — playable arcade game
  • Currency System — coins
  • Shop System — interactable NPC shopkeeper with purchasable items
  • Save System — player progress, time of day, inventory, and more

    • Currently Working On:

  • Camping system — placeable tents and campfires from inventory
  • NPC Daily Schedules — pathfinding, persistent positioning, branching dialogue
  • Intro Story — intro cutscene and player waking in the town Inn
  • Worldbuilding — interior scenes (Arcade, homes), town lore, Highspire’s history

    • Prototyping & Future Goals:

  • Quest system overhaul to support branching, tracked goals, and conditional logic
  • More interactive environmental features (caves, hot springs)
  • Turned-based battles
  • Emotional story arcs tied to exploration, memory, and community
